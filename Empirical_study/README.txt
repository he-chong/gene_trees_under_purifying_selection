Please make sure Python 2.7, Biopython, pandas, matplotlib, RAxML have been installed. 
And the CDS dataset can be downloaded from OrthoMaM database http://www.orthomam.univ-montp2.fr/OrthoMaM_v10b8/

Running the Python script autorun.py, sequences will be automatically extracted then calculated the omega value, and finally estimated the gene tree.
The results generated by autorun.py can be compiled by statistics.py, then drawn into plots by draw.py and drawChi.py
